# Astros
Atividade de validação dos pilares de POO

O código cria um modelo simples para representar corpos celestes, usando pilares como herança, encapsulamento e polimorfismo. A classe base, chamada CorpoCeleste, define características comuns a todos os astros, como nome, tipo e um método chamado descrever(). Já as classes Estrela, Planeta e BuracoNegro herdam dessa superclasse e têm seus próprios atributos e maneiras de se descrever, o que permite que cada astro seja apresentado. No programa principal (main.py), há um menu interativo que lista, descreve e adiciona novos corpos celestes.
